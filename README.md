# Planillas

A [Sails](http://sailsjs.org) application.

## Configuracion


  - Clonar de Github

```
git clone git@github.com:juniorUsca/plan.git
cd plan
```

  - Instalar modulos de node

```
npm install
```

  - Instalar componentes de Bower

```
bower install
```

  - Probar el servidor

```
sails lift
```
